*Englisch/English*

Manual:

The game should be playable on your local browser. To start the game, open the index.html file.
If the game doesn't show up correctly, open it in a (local) server environment (e.g. XAMPP).

Operating the menu and level select (kitchen):
Mouse - left click to select

Player controls in-game:
- WAD -> W = jump, A = walk left, D = walk right
- arrow keys -> arrow up = jump, arrow left = walk left, arrow right = walk right
- Spacebar = jump

Misc. controls:
- M = mute music
- N = mute sounds
- R = restart level
- P = back to level select (kitchen)
*Deutsch/German*

Bedienungsanleitung:

Das Spiel sollte sich momentan im lokalen Browser starten lassen, indem Sie die index.html mit einem Doppelklick öffnen.
Falls das Spiel nicht korrekt angezeigt werden sollte, öffnen Sie es in einer (lokalen) Serverumgebung (z.B. mit XAMPP).

Bedienung des Menüs und der Levelauswahl (Küche):
Maus - Linksklick zum Auswählen

Steuerung des Spielers im Spiel:
- WAD -> W = Springen, A = nach Links laufen, D = nach Rechts laufen
- Pfeiltasten -> Pfeiltaste-Oben = Springen, Pfeiltaste-Links = nach Links laufen, Pfeiltaste-Rechts = nach Rechts laufen
- Leertaste = Springen

Weitere Steuerung:
- M = Musik stumm schalten
- N = Geräusche stumm schalten
- R = Level neustarten
- P = zurück zur Levelauswahl (Küche)
